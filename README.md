# Backend Server

## How to run

1. Install dependencies:npm install

2. Start the server:npx ts-node src/index.ts

3. Endpoints:
- `GET /ping` - Check if server is running
- `POST /submit` - Submit a new form entry
- `GET /read?index=<index>` - Read a specific form entry
